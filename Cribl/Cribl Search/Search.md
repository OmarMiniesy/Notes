### General Notes

Used to search through the data.
- Cribl search can query data at rest. Search in place.
- Uses a query language.
- Has visualizations to show information in different formats.
- Available on cloud and using hybrid deployments.

Can search in data lakes, API endpoints of SaaS services, and Cribl [[Edge]] deployments.

There are also **Notebooks**, which can be used to contain multiple queries, notes, and visualizations. 

**Search Methods:**
- **Index-based Search:** Requires data collection, structuring, and formatting for efficient searching.
- **Search-in-Place:** Queries data at its source (hosts or data lakes) without prior collection or indexing.
- **Federated Search:** Queries data across multiple repositories simultaneously.
